# Requirement

```
Truffle v5.1.64 (core: 5.1.64)
Solidity - 0.7.6 (solc-js)
Node v16.0.0
Web3.js v1.2.9
```

## Install Dependencies

`npm i`

## Compile Contracts

`truffle compile`
